  2. "span": Inline Element

    3. The "span" element is used to group inline content. It is a generic container for small portions of content, usually for applying styles or classes to specific text or inline elements.
    It is often used to apply styles like color, font-size, or background to a small part of a sentence or word.

Lists:

  1. "ul"

    2. The "ul" element is used to create an unordered list, where the list items are presented without a specific order
    list items inside a "ul" are marked with bullets (a small round dot)
    Example

    • Item 1
    • Item 2
    • Item 3
  2. "ol"

    3. The "ol" element is used to create an ordered list, where the list items are displayed in a specific order.
    list items inside an "ol" are numbered (1, 2, 3, etc.)
    Example

    1. First item
    2. Second item
    3. Third item
  3. "li"

    4. The "li" element is used to define a list item in both unordered ("ul") and ordered ("ol") lists.
    Each "li" represents a single item in the list.
    The "li" tag must always be nested inside either a "ul" (unordered list) or "ol" (ordered list).

Forms:

  1. "form"

    2. The "form" element is used to define an HTML form. It contains various input elements that allow the user to submit data to a server or perform an action.

  2. input type="text": Text Input Field

    3. The "input" element is used to create various types of input fields, such as text boxes, checkboxes, radio buttons, etc. The type attribute defines the specific type of input.
    : When type="text", it creates a text input field where the user can type a single line of text.
